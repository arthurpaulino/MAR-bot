# MAR-bot

My bot for the [Much Assembly Required](https://muchassemblyrequired.com) game.

Press `ENTER` to go to *manual mode*. In *manual mode*, press `ESC` to go to *auto mode*.

## Manual mode

* `2` goes north
* `q` goes south
* `1` goes west
* `w` goes east
* `e` to collect biomass
* `d` to drill
* `c` to clean inventory
